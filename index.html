Sistema de Inscrição e Acompanhamento Desportivo

ISANUS

Sistema de Gestão Psicossocial

Plataforma de gestão e acompanhamento psicossocial destinada a escolas de ensino público. O projeto envolveu desde a criação de regras de negócio até a implementação final. Sistema de Banco de Dados postreSQL multi tenant, com autenticação e autorização, construção de API RESTful.

IA Microservice

API e Microsserviço de Inteligência Artificial

Microsserviço construído para disponibilizar diferentes modelos de IA para consumidores. Inclui gerenciamento de uso, processamento, limite de tokens e controle do grau de pensamento computacional.

Meus Assistentes de IA

Converse com diferentes modelos de Inteligência Artificial. Eles funcionam como verdadeiros "secretários" de todos os meus projetos.

Sonnet: Modelo com 64k tokens de saída e 32k para processamento de entrada. É o modelo mais demorado, mas com a resposta mais detalhada.

Nova: Modelo mais rápido com latência otimizada, porém com limite de 16k tokens para entrada e saída. Ideal para conversas curtas e objetivas.

Haiku: Modelo otimizado para respostas rápidas e concisas, pronto para responder a qualquer pergunta.
